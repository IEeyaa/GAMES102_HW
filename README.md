#### 项目环境

| 相关配置 | 我的版本    |
| -------- | ----------- |
| 解释器   | Python 3.10 |
| 虚拟环境 | venv        |



#### 基本配置

Clone项目

```
git clone https://github.com/IEeyaa/GAMES102_HW.git
```

进入文件夹

```
cd GAMES102
```

安装所需依赖

```
pip install -r requirements.txt
```

启动相关项目

```
python xx.py
```
